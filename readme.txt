== Description ==

This is a simple plugin where you can create vacancies. Also users can send their resumes.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPres.
1. Go to Simple Job Manager in WordPress admin area.
1. Simply add the shortcode to page. Warning: You can add only one shortcode to page
